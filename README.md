# Customer Demographics Analytics Dashboard 📊

## Deskripsi Proyek
Proyek ini merupakan tugas untuk memenuhi kelulusan kelas Belajar Fundamental Analisis Data di Dicoding. Proyek ini bertujuan untuk melakukan analisis data pada data pelanggan untuk menjawab pertanyaan bisnis terkait distribusi geografis pelanggan di tingkat negara bagian (State), kota (City), dan wilayah (Region).

## Struktur Direktori
- **/dashboard**: Berisi file dashboard utama (`dashboard.py`), data bersih (`main_data.csv`), dan daftar library (`requirements.txt`).
- **/data**: Berisi dataset mentah yang digunakan dalam analisis.
- **notebook.ipynb**: File analisis data lengkap mulai dari Wrangling, EDA, hingga Visualisasi.
